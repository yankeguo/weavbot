A lightweight personal AI assistant framework.

## Installation

```bash
uv tool install git+https://github.com/yankeguo/weavbot.git
```

Verify:

```bash
weavbot --version
```

**Full documentation:** [https://yankeguo.github.io/weavbot/](https://yankeguo.github.io/weavbot/)
